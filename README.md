# 엔트리-아두이노

본 서는 엔트리에 대한 기본 지식은 있으나, 아직 아두이노 C++코딩에는 입문하지 못한,  **엔트리 블록코딩을 통해 아두이노 기본을 익히고자 하는 분/교육하고자 하는 분**들을 위해 씌여졌습니다.
